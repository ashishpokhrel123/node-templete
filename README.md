# PatientMangementSystem

     Tech Stack Used
       1. Node Js (Backend)
       2. React Js (Frontend)
       3. Google Bucket For Image Storage.
       
# For Setup 
git clone  https://github.com/ashishpokhrel123/PatientMangementSystem

 # (Backend)
    1. cd api/src
    2.  npm install
    3.  create google storage bucket
    4. npm start



# (frontend)
  1.  cd client/pms
 2.  npm install
 3.   npm start
